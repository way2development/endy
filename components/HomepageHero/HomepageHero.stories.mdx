# Homepage Hero (Default)

The default homepage hero displays during off-promotion times. It features a main background image, title, and a CTA button linking to any internal page on the store. On desktop, it also displays an aside column that can be setup to show a background image, a badge image, and a secondary link. The entire aside column acts as the linked element.
