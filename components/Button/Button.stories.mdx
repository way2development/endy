# Button

There are two types of buttons:

- [Button (Primary/Secondary/Disabled)](#primary)
- [Text Button](#text-button)

## Usage

Clicking on a button triggers an action (e.g. opens or closes a modal).

Button has three possible states:

- [Primary](#primary)
- [Secondary](#secondary)
- [Disabled](#button-v5---disabled)

# Inline Button

## Usage

- Inline buttons look like links, but are actually buttons
- Clicking on an inline button triggers an action (e.g. opens or closes a modal).
- Inline buttons do not have an arrow symbol next to it.
- Inline Buttons are meant to be used as a part of a paragraph and will take the font size, font family, and font color of the parent, please use the Block-Inline variants if you are looking for a standalone Button that looks visually the same

### Inline Button v1

export const InlineTemplate = (args) => (
  <Text variant='largeBody' color='gravy'>
    *Offer valid until 05/10/2022 at 11:59PM PST <Button {...args} />
