# Blog Prefooter

- This module allows the author to display different Endy products throughout the blog to advertise to customers
- On every blog page, the author is able to customize which products they want to display, the accompanying product images and the title of the module
- Sale touchpoints will also be displayed whenever the products in this module are on sale
