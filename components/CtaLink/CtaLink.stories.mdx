# CTA Link

There are two types of CTA links:

- [Button Link](#button-link)
- [Text Link](#text-link)

# Button Link

## Usage

- Button links look like buttons, but are actually links
- Clicking on a button link re-directs to an internal or external URL.
- Can open in the same tab or a new tab.

Button links have two possible states:

- [Primary](#primary)
- [Secondary](#secondary)

# Text Link

## Usage

- Clicking on an block-line link re-directs to an internal or external URL.
- Can open in the same tab or a new tab.
- block-line links have an arrow symbol next to it.

# Inline Link

## Usage

- Inline links do not have an arrow symbol next to it.
- Inline links are meant to be used as a part of a paragraph and will take the font size, font family, and font color of the parent, please use the Block-Inline variants if you are looking for a standalone link that looks visually the same

### Inline Link v1

export const InlineTemplate = (args) => (
  <Text variant='largeBody' color='gravy'>
    *Offer valid until 05/10/2022 at 11:59PM PST <CtaLink {...args} />
  </Text>
)

<Canvas>
  <Story
