# Write A Review

This module appears on the reviews page and the product detail pages and is used to allow users to submit a review for any product(s) purchased from endy.com. The products are ranked according to the cross sell ranking.

Fields checked off are required:

- [x] Rating
- [x] Endy Product (title, id and url)
- [x] Email
- [x] Review Title
- [x] Review Content
- [x] Name
- [ ] Location
- [ ] Sleeping Position Preference
