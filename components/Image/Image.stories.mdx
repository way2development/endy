# Image

This component displays an HTML image with an optional alt text rendered with 3 different sourcesets at mobile: 0 - 767, tablet: 768 - 1024, desktop: 1025+. This is commonly used to display lifestyle images.

## Default

srcWidths prop is always required and it is unique for every image. It is used to dictate the width of the image we need to source from Sanity at each of the 2 breakpoints (excluding the largest breakpoint as it will take the width of the uploaded image). The srcWidths values can be found in the mockup by inspecting the width of the image at each of the breakpoints.

<Canvas>
  <Story
    name='Default'
    args={{

## Image with mobile asset

Mobile and tablet assets are optional. If no tablet asset is provided desktop asset is used instead. If no mobile asset is provided tablet asset is used instead.
