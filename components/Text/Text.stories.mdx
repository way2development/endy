## Inline Text

Inline display renders text in a span rather than a paragraph tag. It is not available for headings.

<Canvas>
  <Story
    name='Inline Text'
    args={{
      display: 'inline',
      variant: 'smallBody',
      color: 'gravy',
      text: 'This is '
    }}
  >
    {({ text, ...args }) => (
      <>
        <Text {...args}>{text}</Text>
        <Text display='inline' variant='largeBody'>
          Inline
        </Text>
      </>
    )}
  </Story>
</Canvas>
