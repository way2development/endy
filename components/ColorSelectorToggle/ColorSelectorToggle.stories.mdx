# Color Selector Dropdown

This atom component is used to toggle the color selector dropdown which is currently being used on the shop module for add-on products.

This atom is used in conjunction with the Color Selector Dropdown component. These two components are used together to create the Color Selector component. They live as separate atoms so that they can be used in other components freely.

export const ToggleButton = (args) => {
  const [isExpanded, setIsExpanded] = useState(false)
  return (
    <div style={{ display: 'flex', justifyContent: 'flex-end' }}>
      <ColorSelectorToggle
        setIsExpanded={setIsExpanded}
        selectedColor={'Alphine White'}
        isExpanded={isExpanded}
        locale={'en'}
      />
    </div>
  )
}
