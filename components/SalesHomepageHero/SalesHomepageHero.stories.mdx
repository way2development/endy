# Homepage Hero (Sales)

During active promotions, the homepage hero is customized to promote the current sale offer. This modules displays all relevant sale details and allows the author to customize the homepage hero design. The heading and subcopy can either be text or a lockup image with alt text.

There are currently two sale design layouts:

- Default/Sale Split Screen & Sale/Sale Split Screen
- One Block & One Block Faded
